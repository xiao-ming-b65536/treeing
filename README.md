# Treeing

Generate real directories and empty files from ASCII tree text (e.g. `tree` command output).

This repository receives **clean release snapshots** from private development. Each release is a single commit tagged `vX.Y.Z`.

## Status

Scaffold only — the first full release will be published with:

```powershell
# From the private dev repo (project/)
.\scripts\publish_release.ps1 -Locale en -Version 1.0.0 -Commit -Push
```
